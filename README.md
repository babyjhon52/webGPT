# BMSTUGPT Frontend

Фронтенд зачетного проекта: минималистичный AI-чат с авторизацией, регистрацией,
основным меню, списком чатов, созданием и удалением диалогов, выбором модели и
настройками аккаунта с системным промптом.

## Запуск

```bash
python3 main.py
```

После запуска откройте:

```text
http://127.0.0.1:8000
```

Если порт `8000` занят, сервер автоматически выберет следующий свободный порт и
напишет точный адрес в терминале.

Сейчас данные хранятся локально в браузере через `localStorage`, а ответ
ассистента формируется заглушкой в `script.js`. Backend можно подключить в
функции `generateAssistantReply` или заменить обработчик отправки сообщения на
запрос к своему API.
